# masawada/tap

Homebrew formulae for tools maintained by masawada.

## Usage

```
$ brew install masawada/tap/<formula>
```

Or tap first, then install:

```
$ brew tap masawada/tap
$ brew install <formula>
```

## Formulae

- [mdp](https://github.com/masawada/mdp)
- [macos-hid-inspector](https://github.com/masawada/macos-hid-inspector)
